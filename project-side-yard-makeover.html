Side Yard Makeover

Side yards are the most underused part of any property. This one was a mess — overgrown, uneven, basically dead space nobody went into. We cleared it out, dug a trench along the property line, put in a small segmental block wall to create a clean edge and hold back the neighbour's grade, and finished the whole run with drain rock. Tight space to work in, but it ended up being a clean, functional strip that actually gets used now.
